Online Retail Sales Analysis Database 

Project Overview
I made a simple SQL-based system for this project to help me understand and analyse data on retail sales. The main idea is to put raw sales data into a structured format and then use SQL queries to get useful information from it. This helps us learn how products work, how customers act, and how money is made over time.

Database Design
I made four related tables to help me keep track of the data:
Customers:keeps track of basic information about customers
Products:has information about products, like their name, category, and price
Orders:keeps track of every order that customers make
Order_Items connects products to orders and keeps track of how many of each item there are.
These tables are connected using relationships to avoid duplication.

Objectives
Store structured retail data 
Use SQL queries to perform analysis
Get useful business insights

Key Features
● Find top-selling products
● Identify most valuable customers
● Monthly revenue calculation
● Category-wise sales analysis
● Detect inactive customers
